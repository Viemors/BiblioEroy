    Esse projeto tem como objetivo a criação de um sistema para uma biblioteca, buscando atender suas principais demandas: a navegação entre os livros oferecidos e o empréstimo
    e devolução das obras. Auxiliando os dois principais usuários: o bibliotecário (para administração da biblioteca) e os leitores (para usufruir dos livros disponíveis).

    Esse projeto foi desenvolvido por um grupo de estudantes do segundo ano de Informática Integrada ao Ensino Médio - IFSP, composto por:

    Vivian S. Carvalho, Beatriz Barreto, Isabelli Pereira da Nóbrega, Rebeca Rodrigues, Nátali P. Ribeiro.

    Onde ele foi repartido em duas partes: Back-End (responsável por toda a parte lógica do programa) e o Front-End ( Responsável por toda interface na qual tem contato direto com o usuário, com o consumidor final).

    Front-End: 

    Responsáveis: Isabelli Pereira da Nóbrega, Rebeca Rodrigues. 
    Funções: Toda a construção estética do site, teoria das cores, responsividade, imagens e a idealização das páginas. 
    Utilizaram de forma conjuntas as ferramentas EJS e CSS.

    Back-End: 

    Responsáveis: Vivian S. Carvalho, Beatriz Barreto, Nátali P. Ribeiro.
    Funções: Toda a construção lógica do site, isto é, a criação das funcionalidades do site ( como por exemplo a ação de empréstimo e devolução), a construção e conexão com o banco de dados, o tratamento de erros, toda a estrutura de cadastro, login e criptografia das senhas, entre outros.
    Utilizaram de forma conjuntas as ferramentas EJS, JavaScript e Mysql (sequelize).


    PARA A UTILIZAÇÃO DO SISTEMA:

    1. Para a utilização desse sistema é necessário ter pré-instalado em seu computador o mysql (programa de banco de dados gratuito, que pode ser encontrado para download em: https://www.mysql.com/downloads/).

    1.1 Em seguida é nescessário a criação de um banco de dados com o nome reservado: BioEroy.

    Após esse primeiro passo a passo, é nescessário:

    2.0 Estar logado via VSCode (Visual Studio Code) com sua conta GitHub 

    2.1 A criação de uma nova pasta no seu ambiente de trabalho ( podendo ser feito diretamente pelo VSCode, mas se preferível podendo ser feito também diretamente no computador, e depois aberto no VsCode)

    3.0 Entrar na pasta por meio do Prompt de Comando, ou pelo terminal do próprio VSCode ultilizando o comando "CD", seguindo o modelo:
    C:\Users\usuarioExemplo\Área de Trabalho\Nova pasta\biblioEroy-Pt1>

    3.1 Após isso a ultilização do comando git clone, seguindo o modelo: 
    git clone https://github.com/Viemors/biblioEroy-Pt1.git biblioEroy-Pt1

    4.0 Para a execução do programa é nescessário a ultilização do "código de início" via terminal, sendo:
    node server.js

    5.0 Para a visualização do site, é preciso abrir um navegador web (google, edge) e colocar na barra de navegação http://localhost:3000
